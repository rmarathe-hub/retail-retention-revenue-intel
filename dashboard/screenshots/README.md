# Dashboard Screenshots

PNG exports for the README and portfolio. **Add after building the workbook in Power BI Desktop.**

## Expected files

| Page | Filename | Dashboard page |
|------|----------|----------------|
| 1 | `page1_executive_overview.png` | Executive Revenue Overview |
| 2 | `page2_cohort_retention.png` | Cohort Retention |
| 3 | `page3_rfm_segmentation.png` | RFM Customer Segmentation |
| 4 | `page4_revenue_at_risk.png` | Revenue Concentration & At-Risk |
| 5 | `page5_product_market.png` | Product & Market Performance |
| 6 | `page6_action_plan.png` | Retention Action Plan |

## How to export

1. Build all six pages using [powerbi_dashboard_guide.md](../../docs/powerbi_dashboard_guide.md).
2. In Power BI Desktop: select each page → **File → Export → Export current page as PNG** (or use Snipping Tool).
3. Save with the exact filenames above into this folder.
4. Uncomment or add image markdown in the root [README.md](../../README.md) dashboard section.

## Minimum for portfolio

Recruiters typically need **2–4 strong screenshots**. Priority order:

1. Executive overview (page 1)  
2. Cohort retention heatmap (page 2)  
3. RFM segmentation (page 3)  
4. Action plan summary (page 6)
